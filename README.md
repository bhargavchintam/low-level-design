# low-level-design

Object-oriented / low-level design problems, each in its own folder with the code and a short README
(problem, the classes, the pattern used, how to run). Java and Python, mixed.

Each one has a small demo `main` you can run:

```sh
cd parking-lot && java Main.java
cd rate-limiter && python3 main.py
```

Designs covered: parking lot, elevator, vending machine, ATM, library, tic-tac-toe, snake and ladder,
LRU cache, rate limiter, logging framework, notification service, splitwise, movie ticket booking,
car rental, chess, hotel booking, cache eviction policies, meeting scheduler, model registry,
inference server, mini autograd, tokenizer (BPE), data loader, experiment tracker, vector store,
embedding cache, feature store client, llm rate limiter, prompt template engine, ml pipeline,
ab test assignment, model router, circuit breaker, streaming feature aggregator.

Patterns show up where they fit — strategy, factory, state, observer, chain of responsibility — not forced.
